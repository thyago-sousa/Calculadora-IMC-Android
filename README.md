# Projeto Calculadora IMC - Android Studio (Java)

**Aluno:** Thyago
**Descrição:** Aplicativo desenvolvido para cálculo de Índice de Massa Corporal (IMC). 
O sistema recebe Nome, Peso e Altura, valida os dados e direciona o usuário para 
uma de seis telas específicas de resultado (Abaixo do Peso até Obesidade Grau III).

### Funcionalidades:
- Cálculo preciso de IMC.
- Botão Limpar para resetar o formulário.
- Navegação entre múltiplas Activities via Intent.
- Otimização de RAM (as telas são encerradas ao navegar).
- Exibição de resumo dos dados e mensagens de incentivo no resultado.
